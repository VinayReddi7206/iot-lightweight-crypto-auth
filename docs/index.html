🔁 Tap Lock again on the same message — the binary & code change every time. That's a random salt + IV, so identical messages never produce identical ciphertext.

How it works

  1. You type a message and a secret key, then tap Lock → you get a scrambled code.
  2. Send the code to your friend, and tell them the secret key separately.
  3. Your friend pastes the code, enters the same key, and taps Unlock → they see the message.
  4. Wrong key → it won't open. The message is protected with AES-256 encryption.
